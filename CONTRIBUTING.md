# Contributing

Athenai Transit への貢献に興味を持っていただきありがとうございます。

## Reporting Issues

### Bug Reports

以下を含めてください:

- 再現手順
- 期待される動作と実際の動作
- ブラウザ/OS のバージョン
- スクリーンショット (あれば)

時刻表や停留所名の誤りは、元データ (GTFS/ODPT) に起因する場合があります。

### Feature Requests

機能リクエストは Issue で受け付けています。ユースケースと期待する動作を記載してください。

### Security Issues

セキュリティに関する問題は公開 Issue ではなく、GitHub Security Advisories から報告してください。
