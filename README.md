Transit explorer without destination — discover where you can go from here

目的地を入力しない、お散歩支援型の乗換案内Webアプリ。「今、ここからどこへ行けるか」という偶然の発見と街歩きの楽しさを提供します。

## 主な機能

- **目的地検索なし** -- 「現在地周辺の選択肢」に100%フォーカス
- **エッジマーカー** -- 画面外のバス停/駅の方向を画面端にアイコン表示
- **お散歩地図** -- 地理院地図で街歩きに最適な詳細地形を表示
- **直近3便のボトムシート** -- 次発は「あと〇分」、以降は絶対時刻で表示
- **完全フロントエンド完結** -- APIサーバー不要
